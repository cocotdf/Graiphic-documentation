<h2>Dimension</h2>

<ul>
<li>filters = [n_filters, channel, size[0], size[1], size[2]]</li>
</ul>

<p>The size of filters depends on the input of the <a href="../../../architecture/layers/conv-3d-transpose-add-to-graph/README.md">Conv3DTranspose</a> layer and the parameters n_filters and size.<br/>For example, if the input of the layer has a size of [batch_size = 10, channel = 8, conv_dim1 = 7, conv_dim2 = 5, conv_dim3 = 5], n_filters has the value 6 and size the value [3, 3, 3] then filters will have a size of [n_filters = 6, channel = 8, size[0] = 3, size[1] = 3, size[2] = 3].</p>

<ul>
<li>biases = [n_filters].</li>
</ul>

<p>The size of biases depends on the parameter n_filters of the <a href="../../../architecture/layers/conv-3d-transpose-add-to-graph/README.md">Conv3DTranspose</a> layer.</p>
