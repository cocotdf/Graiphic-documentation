<h1>MaxRoiPool</h1>

<h2>Description</h2>

<p>ROI max pool consumes an input tensor X and region of interests (RoIs) to apply max pooling across each RoI, to produce output 4-D tensor of shape (num_rois, channels, pooled_shape[0], pooled_shape[1]).</p>
