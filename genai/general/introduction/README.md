<h1>Introduction</h1>

<p align="center"><img src="../../../img/genai.svg" alt="GenAI" width="220" /></p>

<p>Welcome to the GenAI Toolkit documentation base.</p>

<p>The GenAI Toolkit extends LabVIEW with a set of building blocks for hosted and local generative AI workflows. The current documentation covers:</p>

<ul>
<li><strong>Anthropic</strong> message-oriented cloud requests for text, image, and PDF prompts.</li>
<li><strong>GGUF</strong> local chat and text-generation sessions, including sampler and session tuning.</li>
<li><strong>GGUF TTS</strong> local text-to-speech initialization, modification, and run flows.</li>
<li><strong>Audio Process</strong> playback helpers to turn generated audio into an end-user experience.</li>
</ul>

<p>The toolkit is designed so you can stay inside LabVIEW while choosing the runtime that best fits your use case: cloud-hosted models, local GGUF models, or a speech pipeline that combines both generation and playback.</p>
