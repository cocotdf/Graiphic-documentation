<h1>Beginner's Guide</h1>

<h2>Choose your first path</h2>

<ul>
<li><strong>Cloud first</strong>: start with <a href="../../anthropic/message/simplify-text-message/README.md">Simplify Text Message</a> to send a single prompt and inspect the returned response cluster.</li>
<li><strong>Local text generation</strong>: start with <a href="../../gguf/text-generation/gguf-init/text-only/session-and-sampler/README.md">Text Only / Session And Sampler</a> to open a local model and validate a first generation flow.</li>
<li><strong>Speech output</strong>: start with <a href="../../gguf/gguf-tts/init/session-sampler/README.md">GGUF TTS Session &amp; Sampler</a>, then finish with <a href="../../audio-process/play/init-playback/play-sound-from-a-buffer/README.md">Play Sound From A Buffer</a>.</li>
</ul>

<h2>Recommended progression</h2>

<ol>
<li>Validate one simple prompt or one simple local session first.</li>
<li>Inspect the main output cluster so you understand what the runtime returns.</li>
<li>Add sampler or session tuning only after the baseline path is stable.</li>
<li>Move to visual, PDF, or audio workflows after text-only validation succeeds.</li>
</ol>
