<h1>Mask</h1>

<h2>Description</h2>

<p>Recopies the Image Src into the Image Dst. If a pixel value is 0 in the Image Mask, the corresponding pixel in Image Dst is set to 0. Type : <em><strong>polymorphic</strong><strong>.</strong></em></p>
