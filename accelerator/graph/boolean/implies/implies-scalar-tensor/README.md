<h1>Implies Scalar Tensor</h1>

<h2>Description</h2>

<p>Negates x and then computes the logical OR of y and the negated x. Both inputs must be Boolean values, numeric values. If x is TRUE and y is FALSE, the function returns FALSE. Otherwise, it returns TRUE. Type : polymorphic.</p>
