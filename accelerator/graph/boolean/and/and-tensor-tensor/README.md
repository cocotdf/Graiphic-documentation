<h1>And Tensor Tensor</h1>

<h2>Description</h2>

<p>Computes the logical AND of the inputs. Both inputs must be Boolean values, numeric values. If both inputs are TRUE, the function returns TRUE. Otherwise, it returns FALSE. Type : polymorphic.</p>
