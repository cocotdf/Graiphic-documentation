<h1>Xor Scalar Tensor</h1>

<h2>Description</h2>

<p>Computes the logical exclusive or (XOR) of the inputs. Both inputs must be Boolean values, numeric values. If both inputs are TRUE or both inputs are FALSE, the function returns FALSE. Otherwise, it returns TRUE. Type : polymorphic.</p>
