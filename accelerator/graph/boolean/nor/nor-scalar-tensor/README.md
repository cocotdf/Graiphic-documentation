<h1>Nor Scalar Tensor</h1>

<h2>Description</h2>

<p>Computes the logical NOR of the inputs. Both inputs must be Boolean values, numeric values. If both inputs are FALSE, the function returns TRUE. Otherwise, it returns FALSE. Type : polymorphic.</p>
