<h1>XNor Tensor Scalar</h1>

<h2>Description</h2>

<p>Computes the logical negation of the logical exclusive or (XOR) of the inputs. Both inputs must be Boolean values, numeric values. If both inputs are TRUE or both inputs are FALSE, the function returns TRUE. Otherwise, it returns FALSE. Type : polymorphic.</p>
