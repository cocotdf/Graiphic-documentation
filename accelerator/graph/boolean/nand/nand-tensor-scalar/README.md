<h1>Nand Tensor Scalar</h1>

<h2>Description</h2>

<p>Computes the logical NAND of the inputs. Both inputs must be Boolean values, numeric values. If both inputs are TRUE, the function returns FALSE. Otherwise, it returns TRUE. Type : polymorphic.</p>
