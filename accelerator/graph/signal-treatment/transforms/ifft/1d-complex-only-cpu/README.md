<h1>1D Complex (only CPU)</h1>

<h2>Description</h2>

<p>Computes the inverse discrete Fourier transform (IDFT) of the input sequence FFT {X}. Warning : This function is only available in CPU. Accepted Dtype FFT {X} : COMPLEX FLOAT/COMPLEX DOUBLE Accepted Rank FFT {X} : 1D (output of FFT with Complex X) Type : polymorphic.</p>
