<h1>1D Complex Optimized (only CUDA)</h1>

<h2>Description</h2>

<p>Computes the inverse discrete Fourier transform (IDFT) of the input sequence FFT {X}. Warning : This function is only available in CUDA. Accepted Dtype FFT {X} : COMPLEX FLOAT/COMPLEX DOUBLE Accepted Rank FFT {X} : 1D (output of FFT with X : 1D even Complex X) Type : polymorphic.</p>
