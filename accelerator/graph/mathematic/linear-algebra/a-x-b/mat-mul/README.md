<h1>Mat Mul</h1>

<h2>Description</h2>

<p>Performs the multiplication of two input matrices or an input matrix and an input vector. Accepted Dtype X Vector | Y Vector : FLOAT/FLOAT16/BFLOAT16/DOUBLE/UINT32/UINT64/INT32/INT64 Type : VI.</p>
