<h1>Outer Product</h1>

<h2>Description</h2>

<p>Computes the outer product of X Vector and Y Vector. Accepted Dtype X Vector | Y Vector : FLOAT/FLOAT16/BFLOAT16/DOUBLE/UINT8..UINT64/INT8..INT64 Type : VI.</p>
