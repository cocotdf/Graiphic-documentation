<h1>Pow Tensor Tensor</h1>

<h2>Description</h2>

<p>Computes x raised to the y power. Accepted Dtype x : FLOAT/FLOAT16/BFLOAT16/DOUBLE/INT32/INT64 y : FLOAT/FLOAT16/BFLOAT16/DOUBLE/INT8/INT16/INT32/INT64 Type : polymorphic.</p>
