<h1>Root Tensor Tensor</h1>

<h2>Description</h2>

<p>Returns the yth root of the input value x. If x is not complex, x must be greater than or equal to zero unless y is an integer. Otherwise, the result is NaN. Accepted Dtype x : FLOAT/FLOAT16/BFLOAT16/DOUBLE/INT32/INT64 y : FLOAT/FLOAT16/BFLOAT16/DOUBLE Type : polymorphic.</p>
