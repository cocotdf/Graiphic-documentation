<p>In this section you’ll find a list of all CuBLAS fonctionalities.</p>

|  | **ICONS** | **DESCRIPTION** |
| --- | --- | --- |
| [amax](../amax-2/README.md) | ![Amax.Png](assets/amax.png) | This function finds the (smallest) index of the element of the maximum magnitude. |
| [amin](../amin/README.md) | ![Amin.Png](/_assets/shared-images/55/55b6c9253c9e-amin.png) | This function finds the (smallest) index of the element of the minimum magnitude. |
| [asum](../asum-2/README.md) | ![Asum.Png](/_assets/shared-images/25/25645e94b737-asum.png) | This function computes the sum of the absolute values of the elements of vector x. |
| [axpy](../axpy-2/README.md) | ![Axpy.Png](/_assets/shared-images/0b/0bdd0814d3d8-axpy.png) | This function multiplies the vector x by the scalar alpha and adds it to the vector y overwriting the latest vector with the result. |
| [copy](../copy/README.md) | ![Copy.Png](/_assets/shared-images/f7/f7d5393de1e7-copy.png) | This function copies the vector x into the vector y. |
| [dot](../dot-2/README.md) | ![Dot.Png](/_assets/shared-images/16/16ab2285588a-dot.png) | This function computes the dot product of vectors x and y. |
| [nrm2](../nrm2-2/README.md) | ![Nrm2.Png](/_assets/shared-images/c8/c8faedc4c64a-nrm2.png) | This function computes the Euclidean norm of the vector x. |
| [rot](../rot-2/README.md) | ![Rot.Png](/_assets/shared-images/43/43bb452be923-rot.png) | This function applies Givens rotation matrix. |
| [rotg](../rotg-2/README.md) | ![Rotg.Png](/_assets/shared-images/fc/fc6bc34a4f2b-rotg.png) | This function constructs the Givens rotation matrix. |
| [rotm](../rotm-2/README.md) | ![Rotm.Png](/_assets/shared-images/c2/c29674554208-rotm.png) | This function applies the modified Givens transformation to vectors x and y. |
| [rotmg](../rotmg-2/README.md) | ![Rotmg.Png](/_assets/shared-images/00/0025b82d1d43-rotmg.png) | This function constructs the modified Givens transformation. |
| [scal](../scal-2/README.md) | ![Scal.Png](/_assets/shared-images/ac/ac8aaba0688b-scal.png) | This function scales the vector x by the scalar alpha and overwrites it with the result. |
| [swap](../swap-2/README.md) | ![Swap.Png](/_assets/shared-images/8c/8cd50f8b291d-swap.png) | This function interchanges the elements of vector x and y. |
| [gemv](../gemv-2/README.md) | ![Gemv.Png](/_assets/shared-images/a8/a8e5335e88e2-gemv.png) | This function performs the matrix-vector multiplication : y = α * op(A)x + β * y |
| [gemm](../gemm/README.md) | ![Gemm.Png](/_assets/shared-images/3d/3dfd94c94605-gemm.png) | This function performs the matrix-matrix multiplication : C = α * op(A) * op(B) + β * C |
