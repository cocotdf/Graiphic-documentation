<h1>Y-th Root of X</h1>

<h2>Description</h2>

<p>Returns the yth root of the input value x. If x is not complex, x must be greater than or equal to zero unless y is an integer. Otherwise, the result is NaN.</p>

<p><strong>Warning : A new tensor is created for the output.</strong></p>
